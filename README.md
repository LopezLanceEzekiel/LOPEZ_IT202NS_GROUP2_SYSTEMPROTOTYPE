# LOPEZ_IT202NS_GROUP2_SYSTEMPROTOTYPE
SECUREVAULT: Integrated File Management System (IFMAS)
SecureVault (also known as IFMAS) is a secure document gateway and file management system designed specifically for academic institutions. Developed as a final project for the Bachelor of Science in Information Technology at Lyceum of the Philippines University-Cavite, the system focuses on streamlining document workflows while maintaining high standards of network and information security.

Project Overview
Academic institutions handle vast amounts of sensitive data, from student records to faculty credentials. SecureVault provides a centralized, authenticated environment to manage these assets, ensuring that only authorized personnel can access, view, or modify institutional documents.

Key Features
Secure Authentication: A dedicated gateway for institutional access with session management and account recovery protocols.

Document Management: Categorized file storage for student handbooks, enrollment reports, curriculum versions, and faculty records.

Centralized Dashboard: A high-level overview of recent uploads, document counts, and system activity.

Student Registry Integration: A built-in tracking system for student status and enrollment data.

Tech Stack & Design
Architecture: Client-Server model focused on Network and Information Security.

Frontend Concept: Modern, dark-themed UI built with HTML5 and CSS3 (utilizing the DM Sans and DM Mono font families).

Security Focus: Single Sign-On (SSO) policy, One-Time Passwords (OTP), and robust session handling as outlined in the project manuscript.

Repository Structure
/docs: Contains the research manuscript and technical documentation.

/ui-prototypes: HTML/CSS wireframes for the Login, Dashboard, and Document Management interfaces.

/src: System source code and implementation details.

Interface Preview
The system utilizes a sleek, dark-mode interface designed for high-productivity environments:

Login Gateway: Minimalist interface for secure institutional entry.

Dashboard: Sidebar navigation with quick access to Documents, Analytics, and Settings.

Management Table: Data-dense views for handling student IDs, names, and enrollment statuses.
